# Logs Directory

This folder is used to store system logs, server error logs, and AI training metrics (like TensorBoard logs).

## Usage
* **FastAPI Server Logs:** Access logs, error tracebacks, and application state changes.
* **TensorBoard Logs:** Output from model training sessions to visualize loss curves and accuracy metrics over time.
* **Prediction Archiving:** Occasional CSV exports of prediction histories for offline analysis.
